🛡️ SafeHer Pro

A modern Women Safety Web Application built using HTML, CSS, and JavaScript.

SafeHer Pro provides emergency assistance features such as SOS alerts, live location tracking, emergency contact management, voice-activated SOS, fake call simulation, safety chatbot, and emergency helpline access.

 🚀 Features

###🚨 Emergency SOS
- One-click SOS activation
- Device vibration feedback
- Automatic location retrieval

### 📍 Live Location Tracking
- Fetches current location using browser geolocation
- Opens location in Google Maps

### 🎤 Voice SOS
- Voice command detection
- Activates SOS when user says:
  - "Help"
  - "Emergency"
  - "Save Me"
 

  ### 📞 Fake Call Feature
- Simulates an incoming phone call
- Useful in uncomfortable situations

### 🔊 Siren Alarm
- Generates a loud emergency alarm sound

### 👥 Emergency Contacts
- Save emergency contacts
- View saved contacts
- Stored using Local Storage

### 🤖 AI Safety Assistant
- Provides instant safety suggestions
- Answers common emergency-related questions

### 🌙 Dark Mode
- Toggle between Light and Dark themes

### 📱 Responsive Design
- Mobile-friendly layout
- Tablet and Desktop support
